API PUBLICA GRATUITA

superhero-api

Multiples datos de superhéroes de Marvel y DC Comics.

¿Qué es un superhéroe?

Los superhéroes son personajes ficticios con habilidades extraordinarias que dedican su vida a proteger al mundo de amenazas y villanos. Estos héroes pueden pertenecer a diferentes universos y ser creados por distintas editoriales, siendo los más conocidos los de Marvel y DC Comics.

¿Qué es la API de Superhéroes?

La API de Superhéroes es una interfaz RESTful de código abierto que permite acceder a información detallada sobre múltiples superhéroes de diferentes universos. Proporciona datos como estadísticas de poder, apariencia, biografía, conexiones y más.

Cómo obtener los datos

Para utilizar la API, se pueden realizar peticiones HTTP a las siguientes rutas:

  • /all.json: Obtén todos los superhéroes en un único archivo JSON.
    Ejemplo: https://akabab.github.io/superhero-api/api/all.json
  • /id/{id.json}: Obtén información completa de un superhéroe específico por su ID.
    Ejemplo: https://akabab.github.io/superhero-api/api/id/1.json
  • /powerstats/{id.json}: Obtén estadísticas de poder de un superhéroe por su ID.
    Ejemplo: https://akabab.github.io/superhero-api/api/powerstats/1.json
  • /appearance/{id.json}: Obtén detalles de la apariencia de un superhéroe por su ID.
    Ejemplo: https://akabab.github.io/superhero-api/api/appearance/1.json
  • /biography/{id.json}: Accede a la biografía de un superhéroe por su ID.
    Ejemplo: https://akabab.github.io/superhero-api/api/biography/1.json
  • /connections/{id.json}: Consulta las conexiones y afiliaciones de un superhéroe por su ID.
    Ejemplo: https://akabab.github.io/superhero-api/api/connections/1.json
  • /work/{id.json}: Obtén información sobre el trabajo y la ocupación de un superhéroe por su ID.
    Ejemplo: https://akabab.github.io/superhero-api/api/work/1.json

Ejemplo de Solicitud Fetch en Node.js 


const fetch = require('node-fetch');

const url = 'https://akabab.github.io/superhero-api/api/all.json';

fetch(url)
  .then(response => response.json())
  .then(data => {
    console.log('Superheroes data:', data);
  })
  .catch(error => {
    console.error('Error fetching superheroes data:', error);
  });
